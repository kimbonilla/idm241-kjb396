Alpha Description

Trigger

Want to Read- Default

  • Hover ON
    • 2 second delay
  • Hover OFF
  • Mouse UP
    • 0.2 second delay

Want to Read- Active

  • Hover ON
    • 2 second delay
  • Hover OFF
  • Mouse UP
    • 0.2 second delay

Rules

Want to Read- Default

  • Hover ON- Cursor changes to finger and background becomes 10% darker
    • 2 second delay- "Add to your to-read shelf." appears as tagline
  • Hover OFF- Cursor changes back to pointer and background changes back to 10% lighter
  • Mouse UP- Button width changes to 55 px and text changes to "saving..."
    • 0.2 second delay- Button width changes to 100 px and content changes to a checkmark with the text "Want to Read"
  • Database record flag set to true

Want to Read- Active

  • Hover ON- Cursor changes to finger
    • 2 second delay- "Remove from your to-read shelf." appears as tagline
  • Hover OFF- Cursor changes back to pointer
  • Mouse UP- Button width changes to 55 px and text changes to "saving..."
    • 0.2 second delay- Button width changes to 80 px and text changes to "Want to Read"
  • Database record flag set to false
