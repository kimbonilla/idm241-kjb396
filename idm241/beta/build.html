"Want to Read" Button

Triggers

Want to Read- Default

  • Hover ON
  • Hover OFF
  • Mouse UP

Want to Read- Active

  • Hover ON
  • Hover OFF
  • Mouse UP

Rules

Want to Read- Default

  • Hover ON
    • Cursor changes to finger
  • Hover OFF
    • Cursor changes to pointer
  • Mouse UP
    • Button width grows to 135 px
    • Content changes to a checkmark with the text "Want to Read"

Want to Read- Active

  • Hover ON
    • Cursor changes to finger
  • Hover OFF
    • Cursor changes to pointer
  • Mouse UP
    • Button width shrinks to 116 px
    • Content changes to just "Want to Read" text

Stars Rating

Triggers

Star Rating- Default

  • Hover ON Star
  • Hover OFF Stars
  • Mouse UP

Star Rating- Active

  • Hover ON Star
  • Hover OFF Stars
  • Mouse UP on Last Orange Star
  • Mouse UP on Star That is Not the Last Orange Star

Rules

Star Rating- Default

  • Hover ON
    • Cursor changes to finger
    • Fills in hovered on Star with yellow along with all the stars in front of it
  • Hover OFF
    • Cursor changes to pointer
    • All stars fade to gray
  • Mouse UP
    • Fills in clicked on star with orange along with all the stars in front of it

Star Rating- Active

  • Hover ON
    • Cursor changes to finger
    • Fills in hovered on Star with yellow along with all the stars in front of it
  • Hover OFF
    • Cursor changes to pointer
    • All stars fade to gray
  • Mouse UP on Last Orange Star
    • Fills in all stars with gray
  • Mouse UP on Star That is Not the Last Orange Star
    • Fills in clicked on star with orange along with all the stars in front of it
    • Fills in the stars after the clicked on star with gray
