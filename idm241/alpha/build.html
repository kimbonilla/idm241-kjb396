Triggers

Want to Read- Default

  • Hover ON
    • 2 second delay
  • Hover OFF
  • Mouse UP
    • 0.2 second delay

  • Hover ON
    • 2 second delay
  • Hover OFF
  • Mouse UP
    • 0.2 second delay

Rules

Want to Read- Default

  • Hover ON
    • Cursor changes to finger
    • Background becomes 10% darker
    • 2 second delay
      • "Add to your to-read shelf." appears as tagline
  • Hover OFF
    • Cursor changes back to pointer
    • Background changes back to 10% lighter
  • Mouse UP
    • Button width changes to 74 px
    • Text changes to "saving..."
    • 0.2 second delay
      • Button width changes to 132 px
      • Content changes to a checkmark with the text "Want to Read"

Want to Read- Active

  • Hover ON
    • Cursor changes to finger
    • 2 second delay
      • "Remove from your to-read shelf." appears as tagline
  • Hover OFF
    • Cursor changes back to pointer
  • Mouse UP
    • Button width changes to 74 px
    • Text changes to "saving..."
    • 0.2 second delay
      • Button width changes to 115 px
      • Text changes to "Want to Read"
